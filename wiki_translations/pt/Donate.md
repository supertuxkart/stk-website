..:: Doar para o SuperTuxKart ::..

{% end_liquid %}

Se desfrutaste do SuperTuxKart e gostarias de ajudar o projeto, ou só dizer obrigado, poderás fazer um donativo para nos ajudar! O SuperTuxKart usa agora o itch.io para receber donativos. Clica em “Download Now” (Descarregar) e poderás escolher o valor que desejas contribuir.

Os donativos ajudam-nos a melhorar o jogo, pois podem ser usados para a compra de hardware, serviços de alojamento e distribuição de ficheiros, servidores para hospedar websites, pagamento a artistas para criar arte melhorada, e muito mais. Podes saber mais sobre a nossa política [aqui](Donation_Policy).

**Não hesites em contactar-nos caso tenhas alguma questão ou problema!**

..:: Pacote de Oferta ::..

{% end_liquid %}

Juntamente com o SuperTuxKart 1.5, há um pacote de oferta que inclui duas pistas desenvolvidas para o SuperTuxKart Evolution — Drainage Dash e Freytra Peaks — disponível para donativos de 5$ ou mais.

{% gallery widths=90%
/assets/wiki/1.5_gift_package.jpg
%}

Para instalar o pacote, primeiro obtém o "1.5 Gift Package" (Pacote de Oferta 1.5) no itch.io, descompacta o ficheiro e segue as instruções incluídas no README. O procedimento segue a [instalação manual de extensões](Installing_Add-Ons).

Infelizmente, por agora não conseguimos oferecer um procedimento de instalação integrado no jogo nem um método simples de instalação para utilizadores Android. Estamos conscientes desta limitação, mas não quisemos adiar o lançamento da versão 1.5 para todos os utilizadores só para a resolver.
